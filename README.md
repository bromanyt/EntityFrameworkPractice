# EntityFrameworkPractice
В этом проекте я практикуюсь в рефлексии, ASP.NET Core MVC, организации CRUD операций с использованием ORM фреймворка.

Проект был командным, моя часть отвечала за подключение к базе данных, созданию контекста (на 80% генерировался автоматически), написанию контроллеров, биндера и всего, что касалось CRUD операций.
